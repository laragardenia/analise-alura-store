# 📊 Alura Store - Análise de Dados

![Status do Projeto](https://img.shields.io/badge/status-concluído-green)

## 📝 Descrição do Projeto

Este projeto foi desenvolvido como um desafio de análise de dados para a rede de lojas Alura Store. O objetivo principal foi processar grandes volumes de dados de vendas para fornecer um embasamento estratégico, identificando qual das quatro unidades da rede possui a menor eficiência operacional e financeira. 

A aplicação utiliza Python e bibliotecas de Data Science para transformar dados brutos em insights sobre faturamento, comportamento de categorias e logística.

## ✨ Funcionalidades e Análises

- **Cálculo de Faturamento:** Processamento automático do total de vendas por unidade.
- **Análise de Produtos:** Identificação de itens com maior e menor giro de estoque.
- **Métricas de Satisfação:** Monitoramento da média de avaliações dos clientes por loja.
- **Estudo Logístico:** Cruzamento entre custo de frete e volume de vendas.
- **Mapeamento Geográfico:** Geração de mapas de calor para visualizar a concentração de vendas por região.

## 🚀 Tecnologias Utilizadas

- **Python:** Linguagem base para toda a lógica de análise.
- **Pandas:** Para manipulação de DataFrames e limpeza de dados.
- **Matplotlib:** Para a criação de gráficos de barras e dispersão.
- **Folium:** Para a geração de mapas interativos de calor (Heatmaps).
- **Google Colab:** Ambiente utilizado para o desenvolvimento do notebook.

## 📂 Estrutura do Repositório

- `AluraStoreBrasil_Lara.ipynb`: Notebook Jupyter contendo o código-fonte e as visualizações.
- `base-de-dados-alurastore/`: Pasta contendo os arquivos CSV com os dados brutos da análise.

## 🚀 Como Executar

O projeto foi construído para ser executado no ambiente Google Colab. Para reproduzir a análise:

1.  Faça o download do arquivo `AluraStoreBrasil_Lara.ipynb`.
2.  Acesse o [Google Colab](https://colab.research.google.com/) e faça o upload do notebook.
3.  Carregue os arquivos CSV contidos na pasta `base-de-dados-alurastore` para o ambiente de arquivos do Colab.
4.  Execute todas as células para visualizar os resultados.

## 💡 Insight Principal da Decisão

Com base na matriz de eficiência, a recomendação final foi a **venda da Loja 4**. Apesar de possuir o frete mais barato (R$ 31.28), a unidade apresentou o menor faturamento da rede (R$ 1,384,497.58), indicando uma falha crítica na conversão de vendas.
